O que é a tabela Cutter?

A tabela Cutter é uma tabela de notação de autor desenvolvida por Charles Ammi Cutter em 1880. Ela é usada para gerar um código numérico que representa as três primeiras letras do sobrenome do autor de uma obra. O código Cutter é usado para organizar as obras de um mesmo autor dentro de um mesmo assunto, facilitando a busca e localização das obras.

A tabela Cutter é organizada da seguinte forma:

  • A primeira letra do sobrenome do autor é representada por um número de um a nove.
  • A segunda letra do sobrenome do autor é representada por um número de zero a nove.
  • A terceira letra do sobrenome do autor é representada por um número de zero a nove.

Por exemplo, o código Cutter para o autor "Barbosa, João" é **112**.

O gerador de notação de autor é uma ferramenta que permite aos bibliotecários e outros profissionais da informação gerar o código Cutter de uma obra de forma automática. O gerador geralmente usa a tabela Cutter para consultar a sequência numérica correspondente às três primeiras letras do sobrenome do autor.

O gerador de notação de autor é uma ferramenta útil que economiza tempo e esforço aos profissionais da informação. Ele também ajuda a garantir que os códigos Cutter sejam gerados de forma consistente, o que é importante para a organização eficaz das coleções de bibliotecas.

Este site é um gerador de notação de autor que usa a tabela Cutter. Ele foi desenvolvido usando HTML, CSS e JavaScript.

O site é simples e fácil de usar. Basta inserir o sobrenome do autor no campo de texto e clicar no botão "Gerar código Cutter". O código Cutter será exibido na tela.
